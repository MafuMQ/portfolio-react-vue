# Portfolio Application Overview

This portfolio application is built using both React and Vue, providing a versatile platform to showcase projects with varying features powered by these popular frameworks.

## Project Features
- **Dual Framework Compatibility**: Users can switch between React and Vue interfaces seamlessly.
- **Responsive Design**: The portfolio is fully responsive, providing a great user experience on desktop and mobile devices.
- **Dynamic Component Loading**: Each framework's components are loaded dynamically, ensuring optimal performance and resource management.

## Setup Instructions
1. **Clone the Repository**:
   ```
   git clone https://github.com/MafuMQ/portfolio-react-vue.git
   cd portfolio-react-vue
   ```
2. **Install Dependencies**:
   Depending on the framework you want to use, run one of the following:
   - For React:
     ```
     npm install --prefix react-frontend
     ```
   - For Vue:
     ```
     npm install --prefix vue-frontend
     ```
3. **Run the Application**:
   - For React:
     ```
     npm start --prefix react-frontend
     ```
   - For Vue:
     ```
     npm run serve --prefix vue-frontend
     ```

## Switching Between Frameworks
To switch from React to Vue or vice versa, you need to stop the current running application and start the other one using the commands provided in the Run section above. Ensure that you run one framework at a time to avoid port conflicts.

Enjoy exploring the portfolio application!
